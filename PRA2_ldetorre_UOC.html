1. Descripción del dataset¶

El dataset que vamos a estudiar es winequality-red.csv, obtenido en https://www.kaggle.com/uciml/red-wine-quality-cortez-et-al-2009, y contiene informacin sobre variantes de vinos rojos portugueses. Gracias a este dataset, podemos intenatr obtener las caractersticas ms importantes de cara a que un vino rojo sea de alta calidad, lo cul podra favorecer y abaratar este tipo de vinos de alta calidad.

Este dataset contiene las siguientes columnas: fixed acidity, volatile acidity, citric acid, residual sugar, chlorides, free sulfur dioxide, total sulfur dioxide, density, pH, sulphates y alcohol. Adems, tiene una ltima columna que es la calidad del vino (entre 0 y 10).

def find_outliers_IQR(df, name):
    q1=df.quantile(0.25)
    q3=df.quantile(0.75)
    IQR=q3-q1
    outliers = df[((df<(q1-1.5*IQR)) | (df>(q3+1.5*IQR)))]
    print(name + ': Num outliers = ' + str(len(outliers)) + '; Min outlier = ' + str(outliers.min()) + '; Max outlier = ' + str(outliers.max()))          
    return outliers

def test_normalidad(df, name):
    k2, p_value = stats.normaltest(df)
    print(name + ': p-valor = ' + str(p_value))          
    return k2, p_value

6. Resolución del problema¶

Hemos visto que la variable que ms influye en la calidad del vino es el alcohol. Adems, tambin hemos comprobado que esta variable es ms influyente en el caso de que la cantidad de alcohol sea superior a 10.

Sin embargo, hemos comprobado que no es suficiente para poder determinar la calidad del vino. Probablemente, una combinacin ms compleja incluyendo alguno de los cidos o sulfuros permitira conseguir un modelo ms preciso de cara a saber la calidad de un vino en base a sus componentes.
